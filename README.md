Movies App: 
  This app will fetch more movies from server and when you click on any item will navigate to show more details about movie clicked.

Technologies:
  Kotlin language,
  MVVM pattern,
  Repository pattern,
  Navigation component,
  REST API (Retrofit),
  Kotlin coroutines,
  Kotlin flows,
  Pagination,
  Dependency injection (Hilt),
  Coil library.
